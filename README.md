## 🎯 项目概述

> **传承千年文化，拥抱智能未来**

舞龙作为中华民族的传统文化瑰宝，承载着深厚的历史底蕴。然而，传统舞龙训练依赖人工经验，缺乏标准化评价体系。本项目创新性地结合深度相机技术与人工智能算法，构建了智能化训练评价反馈系统，旨在推动舞龙运动的数字化传承与科学化教学。
